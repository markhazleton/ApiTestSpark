## Not Archived This Cycle — Corrections to the Raw Scan

The scoring script flagged 20 "orphaned" branding export-variant files under
`.documentation/branding/MakeBoldSolutions/` (CMYK/RGB print and digital
file exports of logo marks, wordmarks, social-media crops, watermark, plus
`SVG and AI/Logo.svg`). An initial attempt to move these into
`.archive/2026-07-16/` was **reverted** after discovering that the entire
`.documentation/branding/` directory is explicitly excluded via
`.gitignore` ("Local source brand pack used to generate app assets").
Moving gitignored, never-committed local assets into `.archive/` (which is
tracked) would have brought previously-untracked binary design files into
git history for the first time — the opposite of the intended effect.
These files were restored to their original location unchanged. No action
is appropriate here; the harvest classification rubric should be extended
in a future iteration to exclude gitignored paths from archive candidacy.
