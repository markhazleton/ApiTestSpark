## Highlights

v1.5.0 completes the Make Bold Solutions brand alignment for API Test Spark. The embedded React SPA now presents API Test Spark as part of the Make Bold Spark product family, using Make Bold Solutions colors, logo assets, Inter Tight typography, refreshed page chrome, and consistent branded focus/accent states.

This release also updates the package-facing identity. NuGet metadata, package README, root documentation, package icon, favicon set, and license attribution now consistently identify Make Bold Solutions as the owner and manager of API Test Spark while preserving Mark Hazleton as an author.

## Changes

### Make Bold Solutions Branding

The app shell, home screen, footer, about/help/config surfaces, explorer accents, and loading states were updated from the previous generic blue theme to the Make Bold Solutions black/red/gray visual language.

NuGet metadata now uses Make Bold Solutions authorship, company, copyright, title, description, and tags. The package icon was regenerated from the Make Bold asset and verified inside the built `.nupkg`.

The site now ships a generated multi-size `favicon.ico`, 16x16 and 32x32 PNG favicons, local Make Bold Solutions logo assets, and the Inter Tight variable font.
